O seu novo escritório: Google Colab

Esqueça instalações complexas. Em Ciência de Dados, o foco é resolver problemas, não brigar com o sistema operacional do seu computador.

O Google Colab é um ambiente Jupyter Notebook hospedado na nuvem. Ele permite que você escreva e execute código Python diretamente no seu navegador, sem precisar configurar nada.

  • Custo zero: O processamento ocorre nos servidores do Google, poupando a memória e a bateria do seu computador.
  • Acesso universal: Seus códigos e análises ficam salvos no seu Google Drive. Você acessa de qualquer máquina.
  • Potência: O Google fornece acesso gratuito a GPUs (Placas de Vídeo) poderosas para quando formos treinar modelos de Inteligência Artificial.

Criando o seu primeiro Notebook

A configuração inicial leva menos de 2 minutos e exige apenas uma conta Google.

  • Acesse o endereço: colab.research.google.com
  • Faça login com a sua conta Google (Gmail ou corporativo).
  • Na janela inicial que se abrir, clique no botão azul "Novo notebook" no canto inferior esquerdo.
  • Pronto! Você verá uma tela branca com uma barra cinza (célula de código). Clique no título superior esquerdo (provavelmente "Untitled0.ipynb") e renomeie para Aula_01_Introducao.ipynb.

Por que Python? E como executar?

O Python é a língua franca dos negócios modernos porque sua leitura é quase como inglês básico. Diferente de linguagens antigas, não há dezenas de caracteres complexos.

No Colab, digite o código abaixo na célula e pressione Shift + Enter (ou clique no botão de "Play" ao lado da célula) para executar e ver o resultado instantaneamente logo abaixo.

# Isso é um comentário. O Python ignora o que vem depois da hashtag.
# Vamos criar variáveis e calcular um faturamento simples

vendas_mes = 150
ticket_medio = 250.50
faturamento = vendas_mes * ticket_medio

print("O faturamento total foi de R$", faturamento)

Células de Código vs Células de Texto

Um grande diferencial dos Notebooks (como o Colab) é que eles são documentos vivos. Você não entrega um arquivo cheio de códigos feios para o seu diretor. Você intercala texto e matemática.

  • Célula de Código: Onde escrevemos Python (cálculos, inteligência artificial, importação de bases).
  • Célula de Texto: Onde escrevemos textos explicativos, títulos e relatórios utilizando a linguagem Markdown.

Atalhos valiosos no teclado:

  • Shift + Enter: Executa a célula atual e pula para a próxima.
  • Ctrl + M + B: Cria uma nova célula de Código abaixo.
  • Ctrl + M + M: Transforma a célula atual em uma célula de Texto.

Aviso Crítico: Onde estão meus arquivos?

Como o Google Colab é uma máquina emprestada do Google, toda vez que você fecha a aba, a máquina é "destruída". Quaisquer arquivos de dados (como planilhas) que você fez upload temporário serão apagados.

Para resolver isso, nós "montamos" o seu Google Drive dentro do Colab. Assim, o Colab consegue ler as planilhas da sua conta de forma segura e permanente.

# Conectando o Colab ao seu Google Drive
from google.colab import drive
drive.mount('/content/drive')

# Ao rodar (Shift+Enter), uma janela pedirá autorização.
# Aceite para dar permissão de leitura ao Colab.

Carregando sua primeira base de dados

Após conectar o Google Drive, podemos usar a biblioteca Pandas para ler planilhas que estão salvas em alguma pasta do seu Drive.

# Importamos a biblioteca Pandas (a mais usada no mundo para dados tabulares)
import pandas as pd

# Lemos um arquivo CSV (nossa base de clientes, por exemplo)
# Lembre-se de ajustar o caminho da pasta onde você salvou no Drive!
